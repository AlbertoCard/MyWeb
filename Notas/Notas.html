¿Qué es HTML?

HTML es un lenguaje de marcado que se utiliza para el desarrollo de páginas de internet. Se trata de la siglas que corresponden a HyperText Markup Language, es decir, Lenguaje de Marcas de Hipertexto.

Se trata de un estándar que sirve de referencia del software que conecta con la elaboración de páginas web en sus diferentes versiones, define una estructura básica y un código (denominado código HTML) para la definición de contenido de una página web, como texto, imágenes, videos, juegos, entre otros.

HTML se escribe en forma de "etiquetas", rodeadas por corchetes angulares (<,>). HTML también puede describir, hasta un cierto punto, la apariencia de un documento, y la mayoría de los estándares actuales permiten una cierta cuantía de "hojas de estilo" que describen la apariencia de un documento separada y externa al mismo.


¿Qué es Semantica?

La semántica es el estudio del significado de las palabras y de las combinaciones de palabras. En el ámbito de la programación web, la semántica se refiere a la forma en que se estructura el código HTML para que sea más comprensible tanto para los humanos como para los navegadores web.
